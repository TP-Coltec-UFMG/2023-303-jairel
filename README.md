# ЖАЙРЕЛ

ㅤㅤUm jogo de ordas com tema Folclore Brasileiro e conta com personas como: <br>
  - Mula sem cabeça
  - Saci Perere
  - Boto cor de Rosa
  - Curupira <br>
  Entre outros... <br></br>
    A ideia inicial é trazer um jogo de layers no estilo ``plants vs zombies`` mesclado com ordas por level no estilo ``swamp attack``
